Introducing The Prayer Request Editor

The purpose of this editor is to allow the creation of a prayer request. It also helps creates the structure for the internal storage of the document.

Usage: The usage for each button is fairly simple. Simply select the text to modify and then click the button and it will automatically chang the text. To make things easier, each of the modification will be shown on the right as a preview.

header - The header button creates the title for the document. There may be more than one header in the document. To use, simply select the text to modify and then click the button

content - The content button wraps around each outline

scripture- The scripture button helps identify the scripture section of the document
